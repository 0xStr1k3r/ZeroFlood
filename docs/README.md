# ZeroFlood Documentation

This folder contains modular documentation for the ZeroFlood DDoS Detection & Mitigation Platform.

## Files

- **index.html** - Main documentation page
- **app.js** - Dynamic markdown loader
- **styles.css** - Dark-themed styling
- **.md files** - Separate documentation sections

## Documentation Sections

| File | Description |
|------|-------------|
| home.md | Welcome & overview |
| installation.md | Installation guide |
| quickstart.md | Quick start guide |
| architecture.md | System architecture |
| detection.md | Attack detection methods |
| mitigation.md | Mitigation system |
| llm.md | LLM integration |
| api.md | API reference |
| configuration.md | Configuration options |
| troubleshooting.md | Common issues |

## View Documentation

```bash
cd docs
python3 -m http.server 8000
# Open: http://localhost:8000
```

Or open `index.html` directly in your browser.
